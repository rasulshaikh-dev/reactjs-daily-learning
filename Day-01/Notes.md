
# Day 01 – React Introduction

## Learning Objective

    Understand what React is, why it is used, and its core concepts.

## What is React?

    React is an open-source JavaScript library developed by Meta for building fast, interactive, and reusable user interfaces using a component-based architecture.

## Why React?

    Build reusable UI components
    Faster rendering with Virtual DOM
    Easy to maintain
    Large ecosystem
    Strong community support
    Excellent for Single Page Applications (SPA)


### 1. Component-Based Architecture

    React applications are built using reusable components.

## 2. Virtual DOM

    React updates only the changed parts of the UI instead of reloading the entire page.

## 3. JSX

    JSX allows developers to write HTML-like syntax inside JavaScript.

## 4. One-Way Data Flow

    Data flows from parent components to child components through props.


## Real-World Uses

    Social Media Apps
    E-commerce Websites
    Dashboards
    Portfolio Websites
    Admin Panels
    Chat Applications
